Introduction to Web Development

Web development is the process of building and maintaining websites. It's the work that happens behind the scenes to make a website look great, work fast and perform well with a seamless user experience. Web developers, or 'devs', do this by using a variety of coding languages. The languages they use depends on the types of tasks they are preforming and the platforms on which they are working.

Front-End Development

Front-end developers are responsible for the parts of a website that people see and interact with. They are typically proficient in HTML, CSS, and JavaScript.

  • HTML
  • CSS
  • JavaScript

Back-End Development

Back-end developers are responsible for the server-side of a website. They manage the database through queries and APIs by client-side commands. This is made possible by using various back-end languages and frameworks.

  • Node.js
  • Ruby on Rails
  • PHP
